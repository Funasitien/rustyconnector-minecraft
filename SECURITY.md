# Security Policy

## Supported Versions

Please be sure that you are running the latest version of RC before reaching out to RC's support.

| Version | Supported          |
| ------- | ------------------ |
| 0.9.x   | :white_check_mark: |
| 0.8.x   | :warning:          |
| < 0.8   | :x:                |

## Reporting a Vulnerability
Please **DO NOT** use the standard issue tracker. Please contact a staff member on discord at https://join.aelysium.group to get started.
